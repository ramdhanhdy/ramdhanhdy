# Ramdhan Hidayat

Background in Physics (BSc) and Data Science (MSc). Currently preparing for PhD-level research while building data pipelines and agentic evaluations.

### Research Interests & Focus Areas
- **Agent Architectures & World Models:** Designing stateful, deterministic multi-agent systems and safety guards. Exploring how models build, update, and leverage internal environment representations.
- **Reinforcement Learning & Alignment:** Investigating reward structures, policy optimization, and systemic biases in language models.
- **Text & Multilingual Networks:** Using graph theory and NLP to map structural relationships across language, rhetoric, and behavior.
- **Human-Centered AI Systems:** Building AI tools that preserve user agency, reduce cognitive load, and turn messy real-world workflows into inspectable systems.

### Selected Projects
- **[Resume Optimizer](https://github.com/ramdhanhdy/resume-optimizer):** LLM job-application pipeline with conversational UX, Supabase-backed state, and provenance/event tracking.
- **[LLM Text Network Analysis](https://github.com/ramdhanhdy/LLM-Text-Network-Analysis):** Graph and NLP pipeline for extracting semantic structure from multilingual text datasets.
- **[SycoBench](https://github.com/ramdhanhdy/SycoBench):** Benchmarking and evaluation work around sycophancy and behavioral tendencies in language models.
